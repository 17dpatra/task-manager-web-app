# Task Manager Web App
This web-application is an assignment for CISC594 at The Harrisburg University of Science and Technology. It uses ReactJS in the frontend and Java + MySQL in the backend.


Preliminary steps to run this project:
1. Make sure you have Java installed on your machine and the PATH is set.
2. Make sure you have Docker on your machine. It is easiest to use the Docker Desktop app.


To run:
1. Start the Docker Desktop app
2. Open a Terminal and navigate to the \backend\authentication\src\main\resources\db folder and run `docker compose up`.
3. Open a Terminal and navigate to the \backend\authentication folder and run `.\gradlew bootRun`.
4. Open a Terminal and navigate to the \frontend folder and run `npm start`.
